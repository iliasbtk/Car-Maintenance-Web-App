HTML

Is a standard markup language for creating web pages, and will be used to define the content of the different screens in our app.

CSS

Cascading Style Sheets will be used to describe the layout of the screens in our app by specifying how the different HTML elements will be displayed in a screen.

JavaScript

a high-level programming language mainly used in web browsers - will be used to control the behaviour of the various App screens in terms of interacting with the user as well as retrieving data from the web (MOT and Road tax information)

JSON

Retrieving data from the web requires using JSON which defines the format of the data to be retrieved by our App.

jQuery Mobile

Phonegap (Cordoba)

APIs

Vehicle Enquiry Service API from DVLA

This will be used to extract the MOT and Road tax status of a given car based on the registration number of the car. In case, this is not possible, we would fall back on manual entering of the last MOT date, and send a reminder accordingly (11 months after the last MOT date). Similarly for road tax: if we are unable to use the Vehicle Enquiry Service API, then the user is prompted to enter the due date for road tax.

Google Maps JavaScript API

In particular, we are going to use the Local Context Library to show the user nearby petrol stations or car garages.
